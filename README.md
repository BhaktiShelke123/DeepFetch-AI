# DeepFetch-AI
Developed a high-performance Retrieval-Augmented Generation (RAG) ingestion pipeline designed to transform unstructured text data into a searchable, high-dimensional vector database. This project focuses on optimizing document granularity and retrieval accuracy using LangChain and OpenAI’s state-of-the-art embedding models.


## 🚀 Getting Started

1. **Clone the repo:**
   git clone https://github.com/BhaktiShelke123/DeepFetch-AI/

3. **Install dependencies:**
   pip install -r requirements.txt

4. **Set up environment variables:**
   Create a .env file and add:
   OPENAI_API_KEY=your_actual_key_here
